# 🌾 FWMS — Farmers & Workers Management System

A full-stack web application connecting farmers with agricultural workers.

## 🚀 Features

- JWT Authentication
- Role-based dashboards (Admin, Farmer, Worker)
- Task management system
- Chat & Rating system

## 🛠 Tech Stack

- Frontend: React + Vite + Tailwind
- Backend: Spring Boot + JWT
- Database: MySQL / H2

## 📁 Structure

backend/ → Spring Boot API  
frontend/ → React App

## ⚡ Run Locally

### Backend

cd backend
mvn spring-boot:run

### Frontend

cd frontend
npm install
npm run dev
